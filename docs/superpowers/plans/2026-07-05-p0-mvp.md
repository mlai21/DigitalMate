# DigitalMate P0 MVP 实现计划

> **面向 AI 代理的工作者：** 必需子技能：使用 superpowers:subagent-driven-development（推荐）或 superpowers:executing-plans 逐任务实现此计划。步骤使用复选框（`- [ ]`）语法来跟踪进度。

**目标：** 按 `docs/prd.md` 的 P0 范围实现可运行的 Web chatbot、长期记忆、联网搜索、基础提醒和最简管理后台。

**架构：** 使用 Next.js App Router 作为 Web/API 层，独立 Node 进程作为常驻 Agent 服务，二者共享 TypeScript Agent Harness 和数据访问层。PostgreSQL + pgvector 作为唯一持久化存储；P0 使用关键词召回，保留向量字段。

**技术栈：** TypeScript、Next.js、React、Tailwind CSS、PostgreSQL、pgvector、Vitest、Playwright、Docker Compose、KIE.AI LLM API。

import path from "node:path";
import { Pool } from "pg";
import { readEnv } from "@/server/config/env";

async function main() {
  const env = readEnv();
  const pool = new Pool({ connectionString: env.databaseUrl });
  const sql = await readFile(path.join(process.cwd(), "src/server/db/schema.sql"), "utf8");
  await pool.query(sql);
  await pool.end();
}

  process.exit(1);
});

export async function* runAgent(input: RunAgentInput): AsyncIterable<string> {
  const memories = await input.repositories.memories.findRelevant(input.userId, input.message);
  const messages = buildMessages({ persona: input.persona, memories, history: input.history, userText: input.message });
