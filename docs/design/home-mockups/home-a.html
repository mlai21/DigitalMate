DDigitalMate

一个有性格、有记忆的
私人数字伙伴

平时像朋友一样陪你聊天、答疑，记得你说过的每件事；逐步成长为能替你完成实际任务的数字员工。部署在你自己的服务器上，数据完全自控。

它会记住重要的事，并在合适的时候主动跟进。

特性

像朋友，更像一位可靠的数字员工

🙂

稳定人设

有名字、有性格、有语气习惯。在所有渠道里，它都是同一个「人」，不是换个皮肤的客服机器人。

📔

长期记忆

自动记住对话中的偏好、事件与关系，越用越懂你。所有记忆条目可随时查看、编辑、删除。

🔍

联网搜索

天气、新闻、实时信息随口一问，答案融进它自己的语气里，而不是甩给你一堆搜索结果。

💬

多渠道同一身份

Web、飞书、钉钉、Telegram……在哪里聊都是同一份记忆、同一个它，跨端无缝续聊。

🌱

自我进化

每天复盘对话、整理记忆，把学到的做法沉淀成技能。新能力经你确认后才生效。

⏰

提醒与主动跟进

「周五提醒我交报销」，到点就来找你。主动消息有频率上限和静默时段，绝不刷屏。

技术

为「长期陪伴」设计的底层

自研 Agent 内核

循环式工具调用、分层记忆、按需加载的技能库，不依赖现成 Agent 框架，为拟人体验量身定制。

分层记忆架构

会话上下文、全量档案、情景记忆、用户画像四层分工——关键事实永远在场，陈年细节随叫随到。

模型无关

统一适配层接入任意 LLM，按用途路由：主对话用能力优先的模型，高频轻量调用用低成本模型。

数据完全自控

部署在自己的 VPS 上，对话与记忆只存自有 PostgreSQL 数据库，敏感信息不写入长期记忆。
